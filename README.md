# StarTrace(ほしのなぞりがき)

星空を指でなぞると、なぞった形に一番近い星座を判定して表示する子供向けWebアプリです。

## セットアップ

```bash
npm install
npm run dev
```

## その他のコマンド

```bash
npm run build   # 本番ビルド
npm run test    # 単体テスト (Vitest)
npm run lint    # ESLint
```
