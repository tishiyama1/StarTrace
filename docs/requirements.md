# StarTrace 要求仕様書

## 1. 概要

**StarTrace** は、子供向けの星座あてWebアプリです。
画面いっぱいに広がる星空を指(またはマウス)でなぞると、なぞった軌跡の「形」に最も近い星座を判定して表示します。

- タイトル案: StarTrace(ほしのなぞりがき)
- 対象: 未就学児〜小学校低学年とその保護者
- 想定デバイス: スマートフォン・タブレットのブラウザ、PCブラウザ(タッチ/マウス両対応)

## 2. 決定事項(ヒアリング結果)

| 項目 | 決定内容 |
|---|---|
| プラットフォーム | Webアプリ(ブラウザ)。インストール不要ですぐ遊べることを優先。 |
| 対象星座 | 有名な星座 10〜15個程度に絞る(88星座すべては対象外)。 |
| 判定方式 | 形状マッチング。なぞった軌跡を正規化し、各星座の形(星を結んだ線)と比較して最も近いものを選ぶ。位置・大きさ・向きが多少違っても許容する。 |
| 追加機能 | 第一段階では実装しない。コア体験(なぞる→判定→結果表示)のみに集中し、演出・豆知識・図鑑機能は将来拡張とする。 |

## 3. 機能要件

### 3.1 コア体験(必須)

1. アプリを開くと、装飾用のランダムな星が散りばめられた夜空の画面が表示される。
2. ユーザーは画面上を指(タッチ)またはマウスでなぞる(1ストロークのドラッグ操作)。
3. なぞっている間、軌跡が光る線としてリアルタイムに描画される。
4. 指を離す(ドラッグ終了)と、なぞった軌跡の形状と、あらかじめ用意された星座テンプレート群の形状を比較し、最も近い星座を1つ判定する。
5. 判定結果として、以下を表示する。
   - 星座名(日本語名 + 学名/英語名)
   - マッチ度(何%近いか、またはスター評価)
   - お手本の星座の形(星をつないだ線)を軌跡と重ねて表示
6. 「もう一度なぞる」ボタンで画面をクリアし、再挑戦できる。

### 3.2 対象星座(12個)

子供に人気・認知度が高く、線でつなぐと特徴的な形になる星座を選定する。

1. おおぐま座(北斗七星)
2. こぐま座(こぐま座の柄杓)
3. カシオペヤ座
4. オリオン座
5. さそり座
6. はくちょう座(北十字)
7. こと座
8. わし座
9. ふたご座
10. おうし座
11. しし座
12. ペガスス座(秋の四辺形)

(詳細データ仕様は `docs/constellation-data.md` を参照)

### 3.3 UI/UX 要件

- 子供向けであることを踏まえ、文字は大きく・平易な言葉・漢字は最小限(ひらがな中心 + ふりがな相当の表記)にする。
- 操作は「なぞるだけ」のワンアクションにし、複雑なメニューは持たない。
- 誤判定でも否定的な表現をせず、常に前向きなフィードバックを返す(例:「◯◯座に近いね!」)。
- レスポンシブ対応(スマホ縦持ち・横持ち・タブレット・PCで破綻しないレイアウト)。

## 4. 非機能要件

- 追加のバックエンド・データベースは不要(すべてクライアント側で完結する静的Webアプリ)。
- オフラインでの完全動作は必須要件としない(将来PWA化は拡張候補)。
- 対応ブラウザ: 最新の Chrome / Safari / Edge / Firefox(モバイル含む)。
- 判定はクライアントサイドのJavaScript/TypeScriptのみで完結し、外部APIに依存しない。

## 5. スコープ外(今回は実装しない)

- 正解時のアニメーション・効果音等の演出強化
- 星座の豆知識・神話ミニ解説
- 図鑑・コレクション機能、ローカルストレージへの進捗保存
- 88星座すべての対応、季節切り替え
- ユーザーアカウント・サーバー同期

上記は将来の拡張候補として `docs/architecture.md` に拡張ポイントとして触れる。
